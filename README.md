## How to Run Locally
1. Install XAMPP or WAMP.
2. Place the `CampusHub` folder in `htdocs` (XAMPP) or `www` (WAMP).
3. Import `database/campushub.sql` in phpMyAdmin.
4. Open browser and go to `http://localhost/CampusHub/homepage/index.php`.
